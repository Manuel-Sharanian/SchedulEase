# Appointment Management System

This project is an internal business management web application designed for companies that work with appointments and services.
The system allows administrators and employees to manage service bookings, customers, and financial calculations in one centralized platform.

## Features

- Role-based control (Admin, Manager, User)
- Admin-controlled user registration with email confirmation
- Appointment scheduling and management
- Service management system
- Financial calculation tools for different time periods

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- Microsoft SQL Server
- JavaScript
- Bootstrap

## Purpose

The goal of this project was to build a practical business web application and gain hands-on experience with backend development.

## Project Status

The system is actively used and continuously improved with new features and optimizations.
